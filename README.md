## Install

1. Store this project to local

```
git clone https://github.com/Coli-co/expense_tracker.git
```

2. Switch to project folder

```
cd expense_tracker
```

3. Install npm packages required

```
npm install
```

4. Load seed data

```
npm run seed
```

5. Start project

```
npm run start
```
